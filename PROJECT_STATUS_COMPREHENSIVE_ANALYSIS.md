## 🚀 QUICK START

### For End Users
1. Visit: https://context-aware-caption-optimization-system.onrender.com
2. Click either "Free Mode" or "Pro Mode"
3. Enter your caption
4. Select tone (optional)
5. Click "🧠 Full Analysis (Structured)"
6. View your comprehensive analysis
7. Copy any variations or hashtags

### For Developers
1. Clone: `git clone https://github.com/saifur033/InspiroAI.git`
2. Install: `pip install -r requirements.txt`
3. Run: `python main.py`
4. Call: `POST http://localhost:5000/api/comprehensive_analysis`
5. Use: Integrate into your workflows

3. Add custom emotions
4. Extend language support
